# Agorastates

깃허브 커뮤니티 디스커션 섹션을 참고하여 클론한 솔로 프로젝트입니다.
순수 자바스크립트를 이용하여 SPA로 완성하였고, 컴포넌트 형식으로 추상화하여 작업했습니다.
